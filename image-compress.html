이미지는 어디로 전송되나요?

전송되지 않습니다. 이 페이지는 사용자가 선택한 이미지를 브라우저의 File API와 Canvas 기능으로 읽고, 브라우저 안에서 새 이미지 파일로 다시 만들어 다운로드합니다. 서버 저장, 서버 업로드, 외부 전송 과정이 없습니다.
사진이 서버에 저장되나요?

아니요. 사진은 서버에 저장되지 않습니다. 선택한 파일은 현재 브라우저에서만 읽고 처리합니다.

개인 사진을 넣어도 괜찮나요?

이 도구 자체는 사진을 서버로 보내지 않습니다. 다만 공용 PC에서는 다운로드한 파일이나 브라우저 캐시를 정리하는 것이 좋습니다.

WebP가 가장 좋은가요?

대부분의 웹 사용 목적에서는 WebP가 용량 대비 화질이 좋습니다. 다만 오래된 프로그램 호환성이 필요하면 JPG가 더 편할 수 있습니다.

PNG를 JPG로 바꾸면 배경이 검게 나오나요?

투명 배경 PNG를 JPG로 바꾸면 투명 영역이 흰색 배경으로 합성되도록 처리했습니다.
